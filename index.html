Hardware and software for an escape room «A Beautiful Mind» in Moscow, 2014. The innovation is a centralized architecture with Ethernet-controlled and flashed AVR devices, and a remote (via internet) debugging system. The tasks (see the cover video) are also unique and cute. The quest is now closed (after a few years in business, see this, this, or that for reviews).

EscapeControl: escape room framework with Internet remote control (access via own pagekite server), simple ways to change game logic via the web interface (Blockly/python with code completion), AVR devices flashed/controlled via Ethernet UDP, user-friendly lock-based computing model for waiting for conditions, and support for various hardware (keyboards, LED lamps, Android via UDP) + custom firmware for AVR that can be changed online.
The project was sold to one of the clients in 2019.
