qml-ar library: existing random dot augmented reality markers by Hideaki Uchiyama made into a Qt (C++ and QML) library for Linux and Android. GPU shaders and heuristics (IMU usage, multithreading, caching) allow for 30FPS on Android compared to ~10FPS with the original code running on Linux only.

Presentation of the qml-ar library at the QtDay 2019 in Firenze, Italy

The red dots on paper are recognized by the qml-ar software (in blue) to know where the phone is. The chest is an augmented reality object. Architecture of the qml-ar library showing the data flow. Each component is a C++ class, a QML file, or a physical device. Test run of the learning activity at the International School of Geneva in 2019.
