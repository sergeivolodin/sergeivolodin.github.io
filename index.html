qml-ar library: existing random dot augmented reality markers by Hideaki Uchiyama made into a Qt (C++ and QML) library for Linux and Android. GPU shaders and heuristics (IMU usage, multithreading, caching) allow for 30FPS on Android compared to ~10FPS with the original code running on Linux only.

Presentation of the qml-ar library at the QtDay 2019 in Firenze, Italy

Whitepaper is a work in progress with data from a test run at the International School of Geneva. Stay tuned!

Each component is a C++ class, a QML file, or a physical device. Test run of the learning activity at the International School of Geneva in 2019.

My Master's Thesis at EPFL, 2020-2021 «CauseOccam: Learning Interpretable Abstract Representations in Reinforcement Learning Environments via Model Sparsity». Here we show that simple (in terms of number of edges) neural network-based causal models result in interpretable abstract representations.
Project's code implements learning a sparse causal model of an RL environment from pixels in PyTorch. Please note the resource requirements...

Our paper from the Google Research internship «Resolving Spurious Correlations in Causal Models of Environments via Interventions». Here we define spurious correlations in RL environments and propose ways to deal with them. Paper was presented at the ICLR CLDM 2020 workshop,

An intermediate version of the project with linear environments was presented at the Machine Learning Summer School 2020 .
